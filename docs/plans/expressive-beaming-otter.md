## Часть B: Оптимизация LinkedIn-профиля через Chrome MCP

### Шаг 1: Аудит текущего профиля

**Инструменты:** `tabs_create_mcp` → `navigate` → `read_page` / `get_page_text`

1. Создать новый таб в MCP-группе (`tabs_create_mcp`)
2. Перейти на `linkedin.com/in/me` (редирект на профиль пользователя)
3. Прочитать страницу (`read_page` / `get_page_text`) — собрать:
   - **Имя и Headline** (заголовок под именем)
   - **About** (секция «О себе»)
   - **Experience** (список позиций с описаниями)
   - **Featured** (избранные публикации/статьи)
   - **Skills** (навыки)
   - **Education** (образование)
4. Проскроллить профиль вниз для чтения всех секций

### Шаг 2: Предложить оптимизированные тексты

На основе аудита предложить пользователю для согласования:

**Headline** (до 220 символов) — формула: Роль + Экспертиза + Ценность
- Фокус: AI-стек, enterprise-опыт (банковский сектор), консалтинг и разработка
- Целевая аудитория: коллеги (CTO, системные аналитики) + инвесторы

**About** (до 2600 символов) — структура:
- Hook (1 строка — зацепить внимание)
- Enterprise-опыт (банковский сектор, системный анализ, масштаб)
- Logic Architecture (что делает компания, AI-стек)
- Чем полезен коллегам и инвесторам
- CTA (как связаться)

**Experience** — каждая позиция:
- Чёткое описание роли и зоны ответственности
- Конкретные достижения с метриками где возможно
- Технологический стек и масштаб проектов

### Шаг 3: Применить изменения через Chrome MCP

**Инструменты:** `navigate`, `find`, `computer` (клики), `form_input` (ввод текста)

После согласования текстов с пользователем:
1. Перейти на страницу редактирования профиля
2. Найти и обновить поле Headline (`form_input`)
3. Найти и обновить секцию About (`form_input`)
4. Обновить Experience описания если нужно
5. Сохранить изменения
6. Прочитать финальное состояние профиля для подтверждения

## Ограничения

- Только B2B-контент в профиле — **никаких** упоминаний студенческих услуг
- Не использовать: «DAO», «генерация текстов», «написание нейросетью»
- Использовать: «технологический подход», «собственный AI-стек»
- Тон: экспертный, уверенный — для коллег и инвесторов
