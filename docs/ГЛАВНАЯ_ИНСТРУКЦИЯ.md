## 1. Общая философия автоматизации

### Что можно автоматизировать:
- ✅ Генерация структуры и плана работы
- ✅ Поиск и систематизация источников
- ✅ Составление библиографических описаний
- ✅ Форматирование текста по ГОСТу
- ✅ Создание шаблонов разделов
- ✅ Проверка логической связности
- ✅ Генерация введения и заключения по готовому тексту
- ✅ Парафраз и рерайт черновиков

### Что нельзя автоматизировать полностью:
- ⚠️ Авторский анализ и выводы (требуется ваша редактура)
- ⚠️ Работа с первоисточниками (нужно читать самому)
- ⚠️ Финальная проверка на антиплагиат
- ⚠️ Защита работы

### Принцип работы:
```
[Ваша идея/тема] → [Claude генерирует черновик] → [Вы редактируете] → [Claude форматирует] → [Готовая работа]
```

## 2. Структура курсовой работы

### Стандартная структура для юридической курсовой (25-30 страниц):

```
1. ТИТУЛЬНЫЙ ЛИСТ (1 стр.)
2. СОДЕРЖАНИЕ (1 стр.)
3. ВВЕДЕНИЕ (2-3 стр.)
   - Актуальность темы
   - Степень научной разработанности
   - Объект исследования
   - Предмет исследования
   - Цель исследования
   - Задачи исследования
   - Методологическая основа
   - Структура работы
   
4. ГЛАВА 1. ТЕОРЕТИЧЕСКИЕ ОСНОВЫ (8-10 стр.)
   1.1. Понятие и сущность [предмета]
   1.2. Исторический аспект развития
   1.3. Правовое регулирование
   
5. ГЛАВА 2. ПРАКТИЧЕСКИЕ АСПЕКТЫ (8-10 стр.)
   2.1. Анализ действующего законодательства
   2.2. Проблемы правоприменения
   2.3. Судебная практика
   
6. ГЛАВА 3. ПЕРСПЕКТИВЫ РАЗВИТИЯ (5-7 стр.)
   3.1. Зарубежный опыт
   3.2. Направления совершенствования
   
7. ЗАКЛЮЧЕНИЕ (2-3 стр.)
8. СПИСОК ИСПОЛЬЗОВАННЫХ ИСТОЧНИКОВ (2-3 стр.)
9. ПРИЛОЖЕНИЯ (при необходимости)
```

## 3. Пошаговый алгоритм написания

### ЭТАП 1: Подготовка (30 минут)

**Действие:** Сформулировать тему и собрать начальную информацию

**Промпт для Claude:**
```
Тема моей курсовой работы: "[ТЕМА]"
Направление: юриспруденция, [отрасль права]

Помоги мне:
1. Сформулировать объект и предмет исследования
2. Определить цель и 4-5 задач
3. Предложить структуру работы (2-3 главы)
4. Указать ключевые нормативные акты для изучения
5. Назвать 5-7 ведущих авторов по теме
```

---

### ЭТАП 2: Создание плана (15 минут)

**Промпт для Claude:**
```
На основе следующей информации составь детальный план курсовой работы:

Тема: [ТЕМА]
Объект: [объект]
Предмет: [предмет]
Цель: [цель]

Требования к плану:
- 2-3 главы
- В каждой главе 2-3 параграфа
- Укажи примерный объем каждого раздела в страницах
- Для каждого параграфа укажи ключевые вопросы для раскрытия
```

---

### ЭТАП 3: Написание основной части (2-4 часа)

**Для каждого параграфа используй промпт:**
```
Напиши параграф "[название параграфа]" для курсовой работы по теме "[ТЕМА]".

Требования:
- Объем: [X] страниц (примерно [X*1800] знаков)
- Стиль: научный, юридический
- Обязательно использовать термины: [термины]
- Ссылаться на: [конкретные НПА или авторов]
- Структура параграфа: тезис → аргументация → примеры → вывод

В конце параграфа сделай логический переход к следующему разделу.
```

**Для анализа судебной практики:**
```
Проанализируй судебную практику по вопросу [вопрос].

Включи:
1. Позиции Конституционного Суда РФ (если есть)
2. Позиции Верховного Суда РФ
3. Примеры из практики арбитражных судов или судов общей юрисдикции
4. Выяви тенденции и противоречия

Формат: научный стиль, со ссылками на конкретные дела.
```

---

### ЭТАП 4: Введение и заключение (30 минут)

**Введение (писать ПОСЛЕ основной части):**
```
На основе следующего текста курсовой работы напиши введение.

[Вставить основной текст или краткое содержание глав]

Структура введения:
1. Актуальность (почему тема важна сейчас)
2. Степень разработанности (кто изучал)
3. Объект исследования
4. Предмет исследования
5. Цель работы
6. Задачи (4-5 штук, начинаются с глаголов: изучить, проанализировать, выявить, определить, разработать)
7. Методы исследования
8. Структура работы (краткое описание глав)

Объем: 2-3 страницы.
```

**Заключение:**
```
На основе следующего текста напиши заключение курсовой работы.

[Вставить основной текст]

Требования:
- Показать достижение цели
- По каждой задаче — краткий вывод
- Основные выводы исследования
- Практическая значимость
- Предложения по совершенствованию (если уместно)

Объем: 2-3 страницы.
```

---

### ЭТАП 5: Оформление (30 минут)

**Промпт для создания оформленного документа:**
```
Создай курсовую работу в формате .docx со следующими параметрами оформления:

Требования СПбГУ:
- Шрифт: Times New Roman, 14 пт
- Межстрочный интервал: 1,5
- Поля: левое 3 см, правое 1,5 см, верхнее и нижнее 2 см
- Абзацный отступ: 1,25 см
- Выравнивание: по ширине
- Нумерация страниц: внизу по центру, начиная со 2-й страницы

Содержание работы:
[Вставить весь текст]
```

## 4. Шаблоны промптов для Claude

### 4.1. Универсальный промпт для параграфа

```
Роль: Ты — эксперт по [отрасль права], помогающий студенту юридического факультета СПбГУ.

Задача: Напиши параграф "[название]" объемом [X] знаков.

Контекст:
- Тема курсовой: [тема]
- Этот параграф находится в главе [номер]: "[название главы]"
- Предыдущий параграф был о: [кратко]
- Следующий параграф будет о: [кратко]

Обязательные элементы:
1. Начни с определения ключевого понятия
2. Приведи мнения 2-3 учёных
3. Проанализируй нормативное регулирование
4. Приведи пример из практики
5. Сделай промежуточный вывод

Стиль: академический, без воды, с юридической терминологией.
```

### 4.2. Промпт для анализа НПА

```
Проведи правовой анализ [название НПА] в контексте темы "[тема]".

Структура анализа:
1. Общая характеристика акта (предмет, сфера действия)
2. Ключевые нормы, относящиеся к теме
3. Проблемы правового регулирования
4. Пробелы и коллизии
5. Предложения по совершенствованию

Используй точные цитаты статей с указанием номеров.
```

### 4.3. Промпт для сравнительного анализа

```
Проведи сравнительный анализ правового регулирования [предмет] в России и [страна/страны].

Критерии сравнения:
1. Нормативная база
2. Понятийный аппарат
3. Субъекты правоотношений
4. Механизм реализации
5. Ответственность за нарушения

Представь в виде:
- Текстового анализа (2-3 страницы)
- Сравнительной таблицы

Сделай выводы о возможности заимствования зарубежного опыта.
```

### 4.4. Промпт для списка источников

```
Составь список использованных источников для курсовой работы по теме "[тема]" в соответствии с ГОСТ Р 7.0.5-2008.

Включи:
1. Нормативные правовые акты (10-15 источников):
   - Конституция РФ
   - Федеральные законы
   - Подзаконные акты
   
2. Научная литература (15-20 источников):
   - Монографии
   - Статьи в журналах (Государство и право, Журнал российского права, Правоведение)
   - Диссертации и авторефераты
   
3. Материалы судебной практики (5-10 источников)

4. Электронные ресурсы (3-5 источников)

Оформи каждый источник по ГОСТу.
```

## 5. Требования к оформлению СПбГУ

### Параметры страницы:
| Параметр | Значение |
|----------|----------|
| Формат | A4 |
| Поле левое | 30 мм |
| Поле правое | 15 мм |
| Поле верхнее | 20 мм |
| Поле нижнее | 20 мм |
| Шрифт | Times New Roman |
| Размер шрифта | 14 пт |
| Межстрочный интервал | 1,5 |
| Абзацный отступ | 1,25 см |
| Выравнивание | По ширине |

### Оформление заголовков:
- **ГЛАВА 1.** — прописными буквами, полужирный, по центру
- **1.1.** — строчными буквами, полужирный, с абзацного отступа
- После заголовка точка не ставится
- Между заголовком и текстом — пустая строка

### Оформление сносок:
- Постраничные, нумерация на каждой странице начинается с 1
- Шрифт: Times New Roman, 10 пт
- Межстрочный интервал: 1

### Оформление списка источников:
```
Нормативные правовые акты:
1. Конституция Российской Федерации (принята всенародным голосованием 12.12.1993) // Собрание законодательства РФ. – 2014. – № 31. – Ст. 4398.

2. Гражданский кодекс Российской Федерации (часть первая) от 30.11.1994 № 51-ФЗ // Собрание законодательства РФ. – 1994. – № 32. – Ст. 3301.

Научная литература:
3. Алексеев, С.С. Теория права / С.С. Алексеев. – М.: БЕК, 1995. – 320 с.

4. Иванов, И.И. Проблемы гражданского права / И.И. Иванов // Государство и право. – 2023. – № 5. – С. 45–52.
```

## 6. Работа с источниками

### Обязательные базы данных:
1. **КонсультантПлюс** — нормативные акты, судебная практика
2. **Гарант** — законодательство, комментарии
3. **eLIBRARY.RU** — научные статьи
4. **Dissercat** — диссертации
5. **СПС "Право.ру"** — судебные решения

### Минимальное количество источников:
- Курсовая работа: 25-35 источников
- Из них НПА: 10-15
- Научная литература: 15-20
- Судебная практика: 5-10

### Актуальность источников:
- НПА: действующая редакция
- Научная литература: преимущественно за последние 5 лет
- Судебная практика: за последние 3-5 лет

## 7. Типичные ошибки и как их избежать

### ❌ Ошибка 1: Описательность вместо анализа
**Плохо:** "Статья 15 ГК РФ устанавливает, что..."
**Хорошо:** "Анализ ст. 15 ГК РФ позволяет выявить проблему..., поскольку..."

### ❌ Ошибка 2: Отсутствие авторской позиции
**Плохо:** Перечисление мнений учёных без вывода
**Хорошо:** "Представляется, что наиболее обоснованной является позиция А.А. Иванова, поскольку..."

### ❌ Ошибка 3: Устаревшие источники
**Плохо:** Ссылка на учебник 2005 года
**Хорошо:** Современные монографии и статьи + классические работы как историография

### ❌ Ошибка 4: Нет связи между главами
**Плохо:** Главы существуют отдельно друг от друга
**Хорошо:** Логические переходы, ссылки на предыдущие выводы

### ❌ Ошибка 5: Заключение не соответствует введению
**Плохо:** В заключении выводы, не связанные с задачами
**Хорошо:** Каждая задача из введения → вывод в заключении

## 8. Чек-листы проверки

### Чек-лист структуры:
- [ ] Титульный лист оформлен по образцу
- [ ] Содержание с указанием страниц
- [ ] Введение содержит все обязательные элементы
- [ ] Каждая глава начинается с новой страницы
- [ ] Параграфы логически связаны
- [ ] Заключение соответствует задачам
- [ ] Список источников оформлен по ГОСТу

### Чек-лист содержания:
- [ ] Тема раскрыта полностью
- [ ] Есть анализ НПА
- [ ] Есть анализ научных позиций
- [ ] Есть судебная практика
- [ ] Есть авторские выводы
- [ ] Нет фактических ошибок
- [ ] Терминология используется корректно

### Чек-лист оформления:
- [ ] Шрифт Times New Roman 14
- [ ] Интервал 1,5
- [ ] Поля соответствуют требованиям
- [ ] Нумерация страниц со 2-й страницы
- [ ] Сноски оформлены правильно
- [ ] Объём соответствует требованиям (25-30 стр.)

### Чек-лист перед сдачей:
- [ ] Проверка на антиплагиат (≥70% оригинальности)
- [ ] Вычитка на опечатки
- [ ] Проверка всех ссылок
- [ ] Сверка списка источников со сносками
- [ ] Файл сохранён в нужном формате
