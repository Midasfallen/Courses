# ЧЕК-ЛИСТЫ ПРОВЕРКИ КУРСОВОЙ РАБОТЫ

---

## ЧЕК-ЛИСТ 1: СТРУКТУРА

### Титульный лист
- [ ] Название СПбГУ полностью
- [ ] Факультет и кафедра указаны
- [ ] Тема соответствует заданию
- [ ] ФИО студента, курс, группа
- [ ] ФИО научного руководителя с должностью
- [ ] Город и год

### Содержание
- [ ] Все разделы перечислены
- [ ] Номера страниц проставлены
- [ ] Названия соответствуют заголовкам в тексте

### Введение (2-3 стр.)
- [ ] Актуальность темы
- [ ] Степень научной разработанности
- [ ] Объект исследования
- [ ] Предмет исследования
- [ ] Цель исследования
- [ ] Задачи (4-6 штук)
- [ ] Методологическая основа
- [ ] Структура работы

### Основная часть
- [ ] Каждая глава с новой страницы
- [ ] Параграфы логически связаны
- [ ] Промежуточные выводы в конце параграфов
- [ ] Анализ НПА присутствует
- [ ] Судебная практика есть
- [ ] Мнения учёных приведены
- [ ] Авторская позиция сформулирована

### Заключение (2-3 стр.)
- [ ] Подтверждение достижения цели
- [ ] Выводы по каждой задаче
- [ ] Нет новой информации

### Список источников
- [ ] Не менее 25 источников
- [ ] НПА: 10-15
- [ ] Литература: 15-20
- [ ] Судебная практика: 5-10
- [ ] Оформление по ГОСТ

---

## ЧЕК-ЛИСТ 2: ОФОРМЛЕНИЕ

- [ ] Шрифт Times New Roman, 14 пт
- [ ] Межстрочный интервал 1,5
- [ ] Поля: левое 3 см, правое 1,5 см, верхнее/нижнее 2 см
- [ ] Абзацный отступ 1,25 см
- [ ] Выравнивание по ширине
- [ ] Нумерация страниц со 2-й страницы
- [ ] Сноски: 10 пт, постраничные
- [ ] Заголовки глав: ПРОПИСНЫЕ, по центру
- [ ] Заголовки параграфов: строчные, с отступа

---

## ЧЕК-ЛИСТ 3: КАЧЕСТВО

- [ ] Научный стиль соблюдён
- [ ] Нет личных местоимений
- [ ] Юридическая терминология
- [ ] Нет фактических ошибок
- [ ] Актуальные редакции НПА
- [ ] Современная литература
- [ ] Орфография проверена
- [ ] Пунктуация проверена

---

## ЧЕК-ЛИСТ 4: ПЕРЕД СДАЧЕЙ

- [ ] Антиплагиат ≥70%
- [ ] Файл открывается
- [ ] Содержание = структура текста
- [ ] Сноски = список источников
- [ ] Объём 25-30 страниц
- [ ] Резервная копия есть

---

## ЭКСПРЕСС-ПРОВЕРКА (2 минуты)

1. [ ] Титульный лист — данные верны?
2. [ ] Содержание — страницы совпадают?
3. [ ] Введение — цель + задачи есть?
4. [ ] Заключение — выводы по задачам?
5. [ ] Формат — 14 шрифт, 1.5 интервал?
6. [ ] Антиплагиат — пройден?
