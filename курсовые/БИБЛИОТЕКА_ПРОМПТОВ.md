# БИБЛИОТЕКА ПРОМПТОВ ДЛЯ ЮРИДИЧЕСКИХ КУРСОВЫХ РАБОТ

---

## 1. ПРОМПТЫ ДЛЯ ПЛАНИРОВАНИЯ

### 1.1. Генерация темы
```
Предложи 10 актуальных тем для курсовой работы по [отрасль права].

Требования к темам:
- Практическая значимость
- Наличие достаточной нормативной базы
- Возможность анализа судебной практики
- Дискуссионность в науке
- Реальная возможность написать за [срок]

Для каждой темы укажи:
1. Формулировку темы
2. Почему тема актуальна (1-2 предложения)
3. Основные НПА для исследования
4. Ключевых авторов
```

### 1.2. Формулировка научного аппарата
```
Тема курсовой: "[ТЕМА]"
Отрасль права: [отрасль]

Сформулируй научный аппарат исследования:

1. ОБЪЕКТ исследования (общественные отношения в сфере...)
2. ПРЕДМЕТ исследования (нормы права, регулирующие...; практика применения...; научные концепции...)
3. ЦЕЛЬ исследования (комплексный анализ... / исследование... / разработка предложений...)
4. ЗАДАЧИ (5-6 штук, каждая начинается с глагола: изучить, проанализировать, исследовать, выявить, определить, разработать, систематизировать)
5. МЕТОДЫ исследования:
   - общенаучные (анализ, синтез, индукция, дедукция, системный подход)
   - частнонаучные (формально-юридический, сравнительно-правовой, историко-правовой)

Формулировки должны быть академическими и соответствовать стандартам юридической науки.
```

### 1.3. Структура работы
```
Составь детальный план курсовой работы по теме "[ТЕМА]".

Параметры:
- Объём работы: [25-30] страниц
- Количество глав: [2-3]
- Параграфов в главе: [2-3]

Для каждого элемента структуры укажи:
1. Название главы/параграфа
2. Объём в страницах
3. Ключевые вопросы для раскрытия (3-5 пунктов)
4. Основные источники (НПА, авторы)

Структура должна обеспечивать:
- Логическую последовательность изложения
- Движение от общего к частному
- Баланс теории и практики
- Возможность формулирования выводов
```

## 2. ПРОМПТЫ ДЛЯ НАПИСАНИЯ ВВЕДЕНИЯ

### 2.1. Актуальность темы
```
Напиши раздел "Актуальность темы исследования" для курсовой работы по теме "[ТЕМА]".

Требования:
- Объём: 0,5-1 страница (800-1500 знаков)
- Показать значимость темы с точки зрения:
  * Теоретической важности (научная дискуссия, неразработанность)
  * Практической значимости (проблемы правоприменения)
  * Социальной актуальности (влияние на общество, экономику)
  * Законодательной динамики (изменения в НПА)
- Использовать статистические данные, если уместно
- Стиль: научный, без личных оценок
- Не использовать слово "актуальность" более 2 раз
```

### 2.2. Степень научной разработанности
```
Напиши раздел "Степень научной разработанности темы" для курсовой по теме "[ТЕМА]".

Включи:
1. Общую оценку изученности темы в юридической науке
2. Перечисление ключевых авторов по группам:
   - Классики (советский период): [если применимо]
   - Современные учёные: [основные исследователи]
   - Авторы диссертаций: [если есть]
3. Указание на аспекты, требующие дальнейшего изучения
4. Обоснование необходимости данного исследования

Объём: 0,5 страницы
Не использовать конструкцию "степень научной разработанности" в тексте.
```

### 2.3. Полное введение
```
На основе следующей информации напиши ВВЕДЕНИЕ для курсовой работы:

Тема: [ТЕМА]
Объект: [объект]
Предмет: [предмет]
Цель: [цель]
Задачи:
1. [задача 1]
2. [задача 2]
3. [задача 3]
4. [задача 4]
5. [задача 5]

Структура работы:
Глава 1: [название] - краткое содержание
Глава 2: [название] - краткое содержание
[Глава 3: если есть]

Требования к введению:
1. Объём: 2-3 страницы
2. Последовательность разделов:
   - Актуальность темы
   - Степень научной разработанности
   - Объект исследования
   - Предмет исследования
   - Цель исследования
   - Задачи исследования
   - Методологическая основа
   - Структура работы
3. Стиль: научный, академический
4. Каждый раздел отделён абзацем
5. Связный текст без нумерации внутри разделов
```

## 3. ПРОМПТЫ ДЛЯ ОСНОВНОЙ ЧАСТИ

### 3.1. Теоретический параграф (понятие и сущность)
```
Напиши параграф "[название параграфа]" для курсовой работы по теме "[ТЕМА]".

Тип параграфа: теоретический (понятие и сущность института)

Структура параграфа:
1. ОПРЕДЕЛЕНИЕ понятия:
   - Легальное определение (если есть в законе)
   - Доктринальные определения (3-4 автора с разными подходами)
   - Сравнительный анализ определений
   - Авторская позиция (какое определение наиболее удачно и почему)

2. ПРИЗНАКИ явления:
   - Выделить 4-6 существенных признаков
   - Каждый признак раскрыть (1-2 предложения)

3. СООТНОШЕНИЕ со смежными понятиями:
   - Показать отличие от близких категорий
   - Определить место в системе права

4. ПРОМЕЖУТОЧНЫЙ ВЫВОД:
   - Краткое резюме параграфа (2-3 предложения)
   - Переход к следующему параграфу

Требования:
- Объём: [3-4] страницы (~6000-7000 знаков)
- Сноски: указывать источники в формате "Автор, название, год, страница"
- Стиль: академический
- Использовать конструкции: "представляется", "по мнению", "следует согласиться", "необходимо отметить"
```

### 3.2. Параграф с анализом законодательства
```
Напиши параграф "[название]" с анализом правового регулирования [предмет].

Структура:
1. СИСТЕМА ИСТОЧНИКОВ:
   - Конституционный уровень (статьи Конституции РФ)
   - Законодательный уровень (федеральные законы)
   - Подзаконный уровень (постановления, приказы)
   - Международные акты (если применимо)

2. АНАЛИЗ КЛЮЧЕВЫХ НОРМ:
   - Для каждого важного акта: краткая характеристика и анализ
   - Цитаты статей (кратко, с номерами)
   - Толкование норм

3. ПРОБЛЕМЫ ПРАВОВОГО РЕГУЛИРОВАНИЯ:
   - Пробелы в законодательстве
   - Коллизии норм
   - Неясности формулировок
   - Правовая неопределённость

4. ВЫВОД:
   - Оценка состояния правового регулирования
   - Переход к следующему параграфу

Объём: [4-5] страниц
Стиль: правовой анализ, со ссылками на конкретные статьи
```

### 3.3. Параграф с судебной практикой
```
Напиши параграф "[название]" с анализом судебной практики по вопросу [предмет].

Структура:
1. ПОЗИЦИИ ВЫСШИХ СУДОВ:
   - Конституционный Суд РФ (постановления, определения)
   - Верховный Суд РФ (постановления Пленума, обзоры практики)
   - Высший Арбитражный Суд РФ (если есть исторические позиции)

2. АНАЛИЗ ТИПИЧНЫХ СПОРОВ:
   - Категории дел
   - Типичные обстоятельства
   - Позиции сторон
   - Выводы судов

3. ПРИМЕРЫ КОНКРЕТНЫХ ДЕЛ:
   - 3-5 характерных дел
   - Фабула (кратко)
   - Решение суда
   - Значение для практики

4. ТЕНДЕНЦИИ И ПРОТИВОРЕЧИЯ:
   - Эволюция судебной практики
   - Противоречия между судами
   - Неоднозначные вопросы

5. ВЫВОД:
   - Обобщение практики
   - Выявленные проблемы

Объём: [4-5] страниц
Формат: научный анализ со ссылками на дела
```

### 3.4. Сравнительно-правовой параграф
```
Напиши параграф "[название]" со сравнительным анализом правового регулирования [предмет] в России и зарубежных странах.

Страны для сравнения: [страны]

Структура:
1. ОБОСНОВАНИЕ ВЫБОРА СТРАН:
   - Почему именно эти правовые системы
   - Что полезного можно заимствовать

2. СРАВНИТЕЛЬНЫЙ АНАЛИЗ по критериям:
   | Критерий | Россия | Страна 1 | Страна 2 |
   |----------|--------|----------|----------|
   | Нормативная база | ... | ... | ... |
   | Понятийный аппарат | ... | ... | ... |
   | Субъекты | ... | ... | ... |
   | Механизм реализации | ... | ... | ... |
   | Ответственность | ... | ... | ... |

3. ПРЕИМУЩЕСТВА ЗАРУБЕЖНОГО ОПЫТА:
   - Что работает лучше
   - Какие решения эффективны

4. ВОЗМОЖНОСТЬ ЗАИМСТВОВАНИЯ:
   - Что можно перенять
   - Какие ограничения существуют

5. ВЫВОД:
   - Рекомендации по совершенствованию российского законодательства

Объём: [4-5] страниц
```

### 3.5. Проблемный параграф
```
Напиши параграф "[название]" о проблемах [сфера] и путях их решения.

Структура:
1. ВЫЯВЛЕНИЕ ПРОБЛЕМ (3-5 проблем):
   Для каждой проблемы:
   - Суть проблемы
   - Причины возникновения
   - Последствия
   - Примеры из практики

2. НАУЧНАЯ ДИСКУССИЯ:
   - Позиции учёных по каждой проблеме
   - Предлагаемые решения в литературе

3. АВТОРСКИЕ ПРЕДЛОЖЕНИЯ:
   - Конкретные меры по решению каждой проблемы
   - Обоснование предложений
   - Формулировки изменений в законодательство (если уместно)

4. ВЫВОД:
   - Приоритетность проблем
   - Перспективы решения

Объём: [5-6] страниц
Стиль: аналитический, с чёткой аргументацией
```

## 4. ПРОМПТЫ ДЛЯ ЗАКЛЮЧЕНИЯ

### 4.1. Стандартное заключение
```
На основе следующего содержания курсовой работы напиши ЗАКЛЮЧЕНИЕ:

Тема: [ТЕМА]
Цель: [цель]
Задачи:
1. [задача 1]
2. [задача 2]
...

Основные выводы по главам:
Глава 1: [основные выводы]
Глава 2: [основные выводы]
Глава 3: [основные выводы]

Структура заключения:
1. Подтверждение достижения цели (1 абзац)
2. Выводы по каждой задаче (по 2-3 предложения на задачу)
3. Основные теоретические выводы (3-4 пункта)
4. Практические рекомендации (если есть)
5. Перспективы дальнейшего исследования (1-2 предложения)

Требования:
- Объём: 2-3 страницы
- Не повторять дословно текст работы
- Не вводить новую информацию
- Формулировки должны быть самостоятельными выводами
- Использовать: "в результате исследования установлено", "анализ показал", "представляется обоснованным"
```

## 5. ПРОМПТЫ ДЛЯ РАБОТЫ С ИСТОЧНИКАМИ

### 5.1. Подбор источников
```
Составь список литературы для курсовой работы по теме "[ТЕМА]".

Требования:
1. Нормативные правовые акты (10-15 источников):
   - Конституция РФ
   - Кодексы (по теме)
   - Федеральные законы
   - Подзаконные акты (при необходимости)
   
2. Материалы судебной практики (5-8 источников):
   - Постановления КС РФ
   - Постановления Пленума ВС РФ
   - Обзоры судебной практики
   
3. Научная литература (15-20 источников):
   - Монографии (5-7)
   - Учебники/учебные пособия (3-4)
   - Научные статьи (7-10), приоритет за последние 5 лет
   - Диссертации/авторефераты (2-3)

4. Электронные ресурсы (2-3):
   - Справочные правовые системы
   - Официальные сайты

Указать реальных авторов, реальные издания.
Оформление по ГОСТ Р 7.0.5-2008.
```

### 5.2. Оформление библиографии
```
Оформи следующие источники по ГОСТ Р 7.0.5-2008 для курсовой работы:

[Список источников]

Требования:
- Нормативные акты: название, дата, номер, источник официального опубликования
- Книги: Автор. Название / Автор. – Город: Издательство, год. – Страницы.
- Статьи: Автор. Название статьи / Автор // Название журнала. – Год. – № X. – С. XX–XX.
- Электронные ресурсы: с указанием URL и даты обращения

Расположить в порядке:
1. Нормативные правовые акты (по юридической силе)
2. Материалы судебной практики (по убыванию уровня суда)
3. Научная литература (в алфавитном порядке)
4. Электронные ресурсы
```

## 6. ПРОМПТЫ ДЛЯ РЕДАКТИРОВАНИЯ

```
Отредактируй следующий текст для соответствия требованиям научного стиля юридической работы:

[ТЕКСТ]

Требования к редактуре:
1. Устранить разговорные выражения
2. Заменить личные местоимения на безличные конструкции

6. Сохранить смысл оригинала

## 7. ПРОМПТЫ ДЛЯ СПЕЦИАЛЬНЫХ ЗАДАЧ

### 7.1. Анализ конкретного НПА
```
Проведи детальный анализ [название НПА] для включения в курсовую работу по теме "[ТЕМА]".

Структура анализа:
1. Общая характеристика:
   - Вид акта, дата принятия
   - Предмет регулирования
   - Структура акта

2. Содержание относящихся к теме норм:
   - Перечисление статей с кратким содержанием
   - Цитаты ключевых положений

3. Правовой анализ:
   - Место в системе законодательства
   - Соотношение с другими актами
   - Особенности правового регулирования

4. Критический анализ:
   - Пробелы и недостатки
   - Неясные формулировки
   - Противоречия

5. Выводы для исследования

Формат: готовый текст для вставки в параграф
```

### 7.2. Анализ научной позиции
```
Проанализируй научную позицию [автор] по вопросу [вопрос] из работы "[название работы]".

Включи:
1. Суть позиции автора (изложение своими словами)
2. Аргументы автора
3. Сравнение с другими точками зрения
4. Критическая оценка (согласие/несогласие с обоснованием)
5. Значение для темы исследования

Объём: 300-500 слов
Стиль: научный анализ
```

### 7.3. Формулировка предложений de lege ferenda
```
На основе проведённого анализа сформулируй предложения по совершенствованию законодательства (de lege ferenda) по теме "[ТЕМА]".

Для каждого предложения:
1. Проблема, которую предложение решает
2. Суть предложения
3. Конкретная формулировка изменения:
   "Предлагается изложить статью X в следующей редакции: ..."
   или
   "Предлагается дополнить статью X пунктом Y следующего содержания: ..."
4. Ожидаемый эффект
5. Возможные риски

Количество предложений: 2-3
Формулировки должны быть юридически корректными
```

## 8. БЫСТРЫЕ ПРОМПТЫ

### 8.1. Переход между параграфами
```
Напиши связующий абзац между параграфами:
Предыдущий параграф: "[название]" - о [кратко содержание]
Следующий параграф: "[название]" - о [кратко содержание]

Объём: 2-3 предложения
```

### 8.2. Промежуточный вывод
```
Сформулируй промежуточный вывод по параграфу:
Параграф: "[название]"
